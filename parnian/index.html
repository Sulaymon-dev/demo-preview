Parnian is a multipurpose Admin and Webapp UI Kit based on Utility-first CSS framework TailwindCSS 3+ . It's also includes a multitude of reusable elements, components, layouts, forms, dashboards and webapps that you can use in your project. Parnian is a powerful, ultra responsive, modern, and flexible UI Kit that can be used to build any modern web application, including a SaaS based interface, custom admin panels, dashboard, CRM, CMS, LMS, e-commerce panel and etc.
